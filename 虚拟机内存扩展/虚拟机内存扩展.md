**VMware 设置里把虚拟硬盘扩展到了 65GB**，但是在 Ubuntu 系统里面看到的还是 **52.5GB**，这是因为：

* **VMware 只负责把虚拟硬盘文件（.vmdk）扩展大**，但里面的 **Ubuntu 分区**没有自动扩展。
* 现在的 Ubuntu 根分区（/）还是老大小（52.5GB），所以系统里面看到的容量没有变。

🔧 **解决方法**：需要在 Ubuntu 里扩展分区。

有两种方式：

---

### 方法一：用 `gparted` 图形工具（最简单）

1. 先安装 `gparted`：

   ```bash
   sudo apt update
   sudo apt install gparted
   ```
2. 打开 `gparted`（需要管理员权限）。
3. 选中你的虚拟硬盘，找到根分区 `/` 所在的分区。
4. 把未分配的 12.5GB 空间分配给根分区。
5. 应用更改，重启。

---

### 方法二：命令行扩展

如果不想用图形界面，可以用 `growpart` + `resize2fs`：

1. 安装工具：

   ```bash
   sudo apt install cloud-guest-utils
   ```
2. 查看磁盘分区：

   ```bash
   lsblk
   ```

   假设你的根分区是 `/dev/sda3`。
3. 扩展分区：

   ```bash
   sudo growpart /dev/sda 3
   ```
4. 扩展文件系统：

   ```bash
   sudo resize2fs /dev/sda3
   ```
5. 现在 `df -h` 应该能看到 65G 了。
