
---

### 1. 使用 `tar` 压缩（常用）

```bash
# 压缩为 tar.gz 格式
tar -czvf 压缩包名.tar.gz 文件夹名

# 示例：压缩名为 myfolder 的文件夹
tar -czvf myfolder.tar.gz myfolder
```

* `c`：创建压缩文件
* `z`：使用 gzip 压缩
* `v`：显示压缩过程
* `f`：指定压缩包的文件名

解压：

```bash
tar -xzvf myfolder.tar.gz
```

---

### 2. 使用 `zip`

先确认安装了 `zip`：

```bash
sudo apt-get update
sudo apt-get install zip unzip -y
```

压缩：

```bash
zip -r 压缩包名.zip 文件夹名

# 示例
zip -r myfolder.zip myfolder
```

解压：

```bash
unzip myfolder.zip
```

---

### 3. 压缩比更高（xz）

```bash
tar -cJvf myfolder.tar.xz myfolder
```

解压：

```bash
tar -xJvf myfolder.tar.xz
```
