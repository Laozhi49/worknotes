当你在树莓派（或任何电脑）上运行命令：

```bash
python3 -m http.server
```

它的作用是——**启动一个简易的HTTP服务器（Web服务器）**，把当前目录变成一个**可通过浏览器访问的网页目录**。

---

### 📦 具体功能

1. **启动一个简易Web服务**

   * 默认在 **8000端口** 启动。
   * 会把你执行该命令的**当前文件夹**当作网站的根目录。
   * 浏览器访问：

     ```
     http://<树莓派的IP地址>:8000
     ```

     你就能看到该目录下的文件列表。

2. **用来快速分享文件**

   * 例如你在树莓派的 `/home/pi/files` 目录下执行：

     ```bash
     cd /home/pi/files
     python3 -m http.server
     ```

     然后在局域网另一台电脑的浏览器访问：

     ```
     http://<树莓派IP>:8000
     ```

     就能直接下载或查看 `/home/pi/files` 下的文件。

3. **测试Web前端页面**

   * 如果你有 `index.html` 等网页文件，可以直接用这个命令在本地运行查看效果，而不需要安装Apache、Nginx之类的服务器。

4. **调试或接口测试**

   * 结合 Python 脚本或简单的 REST 请求测试工具时，可以快速启动一个HTTP端口接收或展示内容（比如你自己编写的静态JSON接口文件）。

---

### ⚙️ 可选参数

* 指定端口号：

  ```bash
  python3 -m http.server 8080
  ```

  → 服务运行在 8080 端口。

* 指定目录（Python 3.7+）：

  ```bash
  python3 -m http.server 8080 --directory /home/pi/share
  ```

  → 共享 `/home/pi/share` 目录的内容。

---

### 🔍 小结

| 功能       | 用途         |
| -------- | ---------- |
| 文件共享     | 局域网中方便传输文件 |
| 本地网页调试   | 预览静态HTML页面 |
| 轻量HTTP服务 | 临时测试HTTP通信 |
